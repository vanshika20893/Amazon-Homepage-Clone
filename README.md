# Amazon Homepage Clone

A fully responsive clone of the Amazon homepage built using HTML5 and CSS3. This project focuses on recreating the layout and user interface of Amazon's landing page while applying modern web design principles.

## Features

* Responsive navigation bar with logo, location, search bar, and menu options
* Hero banner section similar to Amazon's homepage
* Product category and shopping panels
* Interactive hover effects for enhanced user experience
* Responsive design optimized for different screen sizes
* Structured footer section with multiple links

## Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid

## Learning Outcomes

* Improved understanding of responsive web design
* Gained hands-on experience with Flexbox and CSS Grid layouts
* Practiced building real-world website interfaces
* Enhanced CSS styling and component structuring skills

## Project Status

Completed as a front-end practice project to strengthen web development fundamentals.
